## 项目启动

1. `star`本项目
2. `git clone https://github.com/lvr1997/vue3-springboot3-admin.git`
3. 使用`IDEA`打开

### 后端

1. 配置`jdk17+`环境
2. 配置`MySQL`

### 前端

先安装`pnpm`

```bash
# 进入到工程目录
cd vue-project
# 安装依赖
pnpm install
# 启动
pnpm dev
```
